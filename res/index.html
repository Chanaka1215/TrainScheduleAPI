Name : Train Scedule API

Version : 1.0.0

Base URL : http://127.0.0.1:3200/

This is the Train Schedule API. To retreve data follow below URL paths

# Path URL Description
1 / Index page(This Page)
2 /stations/all Retreve All records in the Scedule
3 /stations/byname/< Train Name > Retreve records by specific Name
4 /stations/byid/< Train ID > Retreve records by specific Train Number
5 /stations/bystartstation/< Start Station > Retreve records by specific Start Station
6 /stations/bytraintype/< Train Type > Retreve a record by specific Driver Train Type
7 /routes/byavaiday/< Available Day > Retreve a record by specific Available Day
